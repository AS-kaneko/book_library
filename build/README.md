# アイコンファイル

このディレクトリには、アプリケーションのアイコンファイルを配置します。

## 必要なファイル

### macOS
- **ファイル名**: `icon.icns`
- **推奨サイズ**: 512x512px以上
- **形式**: ICNS形式

### Windows
- **ファイル名**: `icon.ico`
- **推奨サイズ**: 256x256px含む複数サイズ (16, 32, 48, 64, 128, 256)
- **形式**: ICO形式

### Linux
- **ファイル名**: `icon.png`
- **推奨サイズ**: 512x512px
- **形式**: PNG形式

## アイコン作成方法

### オプション1: オンラインツールを使用

1. [iConvert Icons](https://iconverticons.com/) にアクセス
2. 1024x1024pxのPNG画像をアップロード
3. すべての形式（ICNS, ICO, PNG）をダウンロード
4. このディレクトリに配置

### オプション2: electron-icon-builderを使用

```bash
# インストール
npm install -g electron-icon-builder

# 変換（1024x1024pxのicon.pngから）
electron-icon-builder --input=./icon.png --output=./build
```

## デフォルトアイコン

アイコンファイルが存在しない場合、Electronのデフォルトアイコンが使用されます。

## デザインガイドライン

- **シンプル**: 小さいサイズでも認識しやすいデザイン
- **コントラスト**: 背景色と明確なコントラスト
- **統一感**: アプリケーションのテーマに合わせた色使い
- **書籍モチーフ**: 図書管理システムのため、本や図書館をイメージさせるデザインを推奨

## 推奨デザイン例

📚 書籍のアイコン
📖 開いた本
🏢 図書館の建物
📋 リストとチェックマーク

背景色は青系または緑系が一般的です。
